<!-- Additional CPython ast nodes needed in this plan -->

List/Dict/Subscript nodes:
  ast.List(elts: list[expr], ctx: ast.Load()|ast.Store())
  ast.Dict(keys: list[expr|None], values: list[expr])
  ast.Subscript(value: expr, slice: expr, ctx: ast.Load()|ast.Store())

Function/Return nodes:
  ast.FunctionDef(name: str, args: ast.arguments, body: list[stmt],
                  decorator_list: [], returns: None)
  ast.arguments(posonlyargs: [], args: list[ast.arg], vararg: None,
                kwonlyargs: [], kw_defaults: [], kwarg: None, defaults: [])
  ast.arg(arg: str, annotation: None)
  ast.Return(value: expr | None)

Method calls (.append, .remove, len()):
  ast.Attribute(value: expr, attr: str, ctx: ast.Load())
  ast.Call(func: ast.Attribute(...), args: list[expr], keywords: [])
  len(): ast.Call(func=ast.Name("len", ast.Load()), args=[target_expr], keywords=[])

Atena node → CPython ast mapping (list/dict/function constructs):
  ListLiteral(elements) → ast.List(elts=[emit(e) for e in elements], ctx=ast.Load())

  DictLiteral(pairs: list[tuple[str, Node]]) →
    ast.Dict(
      keys=[ast.Constant(k) for k, _ in pairs],
      values=[emit(v) for _, v in pairs]
    )

  IndexAccess(target, index, index_converted) →
    ast.Subscript(
      value=emit(target),
      slice=emit(index),   ← emit verbatim; index already 0-based (or _atena_index call) from analyzer
      ctx=ast.Load()
    )
    CRITICAL: never re-shift index. If index_converted is True, emit(index) emits the
    already-shifted value. If index is a FunctionCall(_atena_index), emit it as a call.
    The generator reads and emits; it never re-derives.

  DotAccess(target, name) → (read context)
    ast.Subscript(
      value=emit(target),
      slice=ast.Constant(name),
      ctx=ast.Load()
    )

  DotAccess in dot-write Assign target (write context — see Plan 02 _emit_Assign):
    ast.Subscript(
      value=emit(node._dot_target.target),
      slice=ast.Constant(node._dot_target.name),
      ctx=ast.Store()
    )
    [Already handled in Plan 02 _emit_Assign via hasattr(_dot_target) check]

  ListAdd(target: str, value) →
    ast.Expr(
      ast.Call(
        func=ast.Attribute(value=ast.Name(_mangle(target), ast.Load()), attr="append", ctx=ast.Load()),
        args=[emit(value)],
        keywords=[]
      )
    )

  ListRemove(target: str, value) →
    ast.Expr(
      ast.Call(
        func=ast.Attribute(value=ast.Name(_mangle(target), ast.Load()), attr="remove", ctx=ast.Load()),
        args=[emit(value)],
        keywords=[]
      )
    )

  FunctionCall(name="length", args=[target]) → (special-cased)
    ast.Call(func=ast.Name("len", ast.Load()), args=[emit(args[0])], keywords=[])
    Note: "length" is the Atena keyword; emit as Python "len". Do NOT use _mangle("length")
    since "length" is not a Python keyword — but it maps to "len", not itself.

  FunctionDef(name, params, body) →
    ast.FunctionDef(
      name=_mangle(name),
      args=ast.arguments(
        posonlyargs=[], vararg=None, kwonlyargs=[], kw_defaults=[], kwarg=None, defaults=[],
        args=[ast.arg(arg=_mangle(p), annotation=None) for p in params]
      ),
      body=[emit(s) for s in body],
      decorator_list=[],
      returns=None
    )

  Return(value) → ast.Return(value=emit(value))

_build_preamble() — real implementation:
  Parse helper source strings with ast.parse() and extract the body:
    def _build_preamble(self) -> list[ast.stmt]:
        stmts = []
        if "_atena_index" in self._used_helpers:
            stmts.extend(ast.parse(_ATENA_INDEX_SRC).body)
        if "_atena_concat" in self._used_helpers:
            stmts.extend(ast.parse(_ATENA_CONCAT_SRC).body)
        return stmts
  Order: _atena_index before _atena_concat (arbitrary but deterministic).
  ast.fix_missing_locations is called on the whole module after prepending preamble —
  it fills in locations for preamble nodes too.

FunctionCall with "length" special-case:
  In _emit_FunctionCall, check `if node.name == "length": emit as len(args[0])`.
  "length" is NOT in keyword.kwlist so _mangle won't touch it, but it maps to "len".
  Other special names: "str" → emit as str(...) call (pass-through, already correct).
  "_atena_concat" / "_atena_index" → add to self._used_helpers, emit as plain call.
  All other names → emit as _mangle(name)(args).
