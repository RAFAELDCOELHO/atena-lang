    _current(self) -> str | None:
      Return self._source[self._pos] if self._pos < len(self._source), else None.

    _peek(self) -> str | None:
      Return self._source[self._pos + 1] if self._pos + 1 < len(self._source), else None.

    _advance(self) -> str | None:
      Save current char. Advance self._pos by 1. If the consumed char was '\n', increment
      self._line and reset self._col to 0. Otherwise increment self._col by 1. Return the
      saved char.

    _emit_token(self, tok_type: TokenType, value: str, col: int, source_line: str) -> None:
      Append Token(type=tok_type, value=value, line=self._line, col=col, source_line=source_line)
      to self._tokens.

    tokenize(self) -> list[Token]:
      This plan's implementation is a line-oriented outer loop + per-line inner char scan.
      Plan 03 will restructure the outer loop to add the indentation engine; keep the inner
      scan logic factored into _scan_line_content(raw_line: str) so Plan 03 can call it
      without rewriting the scan code.

      Outer loop: iterate over self._lines (each is a physical line, keepends=True so '\n' is included).
      For now (Plan 02 only): call self._scan_line(raw_line) for each line. After all lines,
      emit EOF: self._emit_token(TokenType.EOF, "", 0, "").
      Return self._tokens.

      _scan_line(self, raw_line: str) -> None:
        Strip the trailing '\n' for source_line display:
          source_line = raw_line.rstrip('\n')  — use this as the source_line arg to all tokens on this line.
        Skip blank and comment-only lines BEFORE any other processing:
          stripped = raw_line.lstrip()
          if not stripped or stripped.startswith('#'): advance self._line, return.
          (In Plan 02 this skip is a no-op stub for the indentation layer — Plan 03 integrates
          blank/comment skip properly. For now, just continue without emitting tokens for those lines.)
        Set a local cursor `i = 0` within the content of the line (after leading whitespace).
        Actually: use the global self._pos cursor. The scan reads characters one at a time using
        _current() / _advance() / _peek(). Because self._lines is pre-split, the global pos
        must track absolute position in self._source. The cleaner approach: implement _scan_line
        to scan using the global pos cursor; the outer loop just calls _scan_line per line.
        (NOTE: the outer loop over self._lines is fine; _scan_line must advance pos past the
        entire physical line including the '\n' newline character at its end.)

      Per-character dispatch in the inner scan loop (while not at '\n' or None):

      Leading whitespace at the start of the line (indentation) — in Plan 02, skip all leading
      whitespace chars without emitting tokens: while _current() in (' ', '\t'): _advance().
      (Plan 03 replaces this with the real indentation engine.)

      Then scan remaining characters until '\n' or None:

      Whitespace mid-line: ' ' or '\t' — _advance(), continue.
      Letter or '_': scan identifier/keyword.
        Collect chars while _current() is alphanumeric or '_'.
        Lookup word in KEYWORDS: tok_type = KEYWORDS.get(word, TokenType.IDENTIFIER).
        Emit token with the collected word.
      Digit: scan integer. Collect digits while _current().isdigit().
        After collecting all digits, check decimal off-ramp per D-02:
          if _current() == '.' and _peek() is not None and _peek().isdigit():
            report error: `'Atena works with whole numbers only — try {integer_part} or {integer_part+1} instead of {integer_part}.{fraction_part}.'`
            Consume the '.' via _advance(). Consume remaining fractional digits via _advance() loop.
            Emit the integer part as a NUMBER token (emit the digits collected before the '.').
          else: emit integer as NUMBER token normally.
      '"' (double quote): scan string literal.
        Save start_col = self._col. _advance() to consume opening '"'.
        Collect chars into buf while _current() not in ('"', '\n', None): buf.append(_current()); _advance().
        If _current() == '"': _advance() (consume closing '"'); emit STRING token with ''.join(buf).
        Else (unterminated): self._errors.add(self._line, 'I found a piece of text that was never closed — make sure every " has a matching ".', source_line). No token emitted. (pos already at '\n' or None; outer loop handles advance.)

          if _current() == '=': _advance() again; emit COMPARISON with value "==".
          else: emit ASSIGN with value "=".
      '!': _advance(); if _current() == '=': _advance(); emit COMPARISON with value "!=".
           else: report unexpected-character error; already advanced one char (always makes progress).
      '<': _advance(); if _current() == '=': _advance(); emit COMPARISON with value "<=".
           else: emit COMPARISON with value "<".
      '>': _advance(); if _current() == '=': _advance(); emit COMPARISON with value ">=".
           else: emit COMPARISON with value ">".
      '+', '-', '*', '/': _advance(); emit OPERATOR with the single char value.

      '[': _advance(); emit LBRACKET with value "[".
      ']': _advance(); emit RBRACKET with value "]".
      '{': _advance(); emit LBRACE with value "{".
      '}': _advance(); emit RBRACE with value "}".

      '.': _advance(); emit DOT with value ".".
