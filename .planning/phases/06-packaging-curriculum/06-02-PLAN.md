  Show a string, a number, and a string-plus-number concatenation.

  One `ask`, one assignment, one `show`. This is interactive — requires canned stdin in tests.

`03-variables.atena` — concept: variables & arithmetic
  Assign two numbers, compute their sum, product, and difference, show each.
  No conditionals or loops. Demonstrates `=`, `+`, `-`, `*`, `/` with show.

`04-conditionals.atena` — concept: `if` / `else`
  Assign a score. If score >= 6: show "Passing". Else: show "Not yet".
  One if/else block, no loop, no functions.

`05-while.atena` — concept: `while` loop
  Count from 1 to 5 using a `while` loop. Show each step or show final count.
  One while loop with a condition and a counter increment. No `repeat`.

`06-repeat.atena` — concept: `repeat N times`
  Repeat a block exactly 3 times using `repeat`. Show a fixed message each iteration.
  One `repeat` block. No while, no condition.

`07-functions.atena` — concept: functions & `return`
  Define a function `double(n)` that returns `n * 2`. Call it with 5, show the result.
  One `function` definition with `return`, one call. No lists, no dicts.

`08-lists.atena` — concept: lists (1-indexed)
  Create a list of three grades. Use `add` to append a value. Use `length` to show the count.
  Show `grades[1]` (the first element) with a comment: "# Lists in Atena start at 1, not 0".
  Use `remove` to remove the added value. Show the updated count.
  MUST include `grades[1]` to demonstrate 1-indexing explicitly (not `grades[0]`).

`09-dicts.atena` — concept: dicts
  Create a dict `person = {name = "Ana", age = 20}`.
  Read a field: `show person.name`.
  Write/update a field: `person.age = 21`.
  Show the updated field with a concatenation: `show "Updated age: " + person.age`.
  MUST show both dot read and dot write.
